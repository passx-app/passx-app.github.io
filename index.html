Frequently Asked Questions

Does PassX collect any personal data?

No. PassX does not any personally identifiable information. The only data collected is anonymous crash reporting, which helps improve the app's performance and stability.

Is my information stored anywhere?

No. Your pass data stays entirely on your device. When you add a pass to Apple Wallet, PassX sends only anonymous metadata to a secure server for cryptographic signing. This data is meaningless and never stored or logged.

Is it safe to use PassX for personal or sensitive cards?

Yes. PassX was built with privacy and security in mind. All processing happens locally, except for the required signing step, which is handled securely. Your passes are never stored on external servers.

Why does PassX require a server to sign passes?

Apple Wallet requires passes to be cryptographically signed with a private key before being accepted. This key must be kept secure and cannot be stored on the device, which is why a minimal amount of anonymous metadata is sent to a signing server. Nothing sensitive is included, and nothing is stored.
